• Ein Framework basierend auf IT: Das Harold-Leavitt Modell

    Das Harold-Leavitt Modell, auch bekannt als das “Leavitt’s Diamond” oder “Organizational Change Model”, ist ein Rahmenwerk zur Analyse von Organisationen und deren Transformation, insbesondere in Bezug auf den Einfluss von Informationstechnologie (IT).
